Lesson 12

Формы-1

        Атрибуты тега form
        action - URL-адре, по которому следует отправить данные формы после ее заполнения
        method - HTTP-метод, который будет использоваться при отправке формы (GET, POST)
        enctype - способ кодирования
        autocomplete - включение автозаполнения формы
        name - название формы
        novalidate - отключение встроенной проверки данных формы на корректность ввода
        target - определяет, где отображать результат отправки формы

        Атрибуты тега input (текстовый ввод)
        text - поле для ввода текста (это значение по умолчанию)
        password - поле ввода пароля
        email - поле ввода вдреса электронной почты
        search - поле ввода поискового запроса
        tel - поле ввода номера телефона
        url - поле ввода url-адреса
        hidden - скрытое поле

        Атрибуты тега input (ввод какой-либо информации)
        date - элемент ввода даты
        datetime-local - элемент ввода локальной даты
        month - элемент выбора месяца
        week - элемент ввода недели
        time - элемент ввода времени
        number - элемент ввода числа
        range - элемент указания диапазона
        color - элемент выбора цвета
        file - элемент выбора файла
        checkbox - флажок переключатель

        Атрибуты тега input (различные кнопки)
        button - кнопка
        submit - кнопка отправки формы
        image - изображение как кнопка отправки формы
        reset - кнопка сброса элементов формы
        radio - радио-кнопка

        Основные атрибуты тега input
        name - название элемента
        placeholder - описание элемента, отображаемое в поле ввода
        required - указывает, что данное поле должно обязательно иметь какое-либо непустое значение перед отправкой формы
        readonly - данное поле предназначено только для чтения
        value - значение элемента
        disabled - указывает, что элемент input должен быть отключен
