        Устаревшие атрибуты, ими лучше не пользоваться, используйте CSS

        align - выравнивание таблицы на странице (left, right, center)
        width - ширина таблицы
        bgcolor - цвет фона таблицы
        border - ширина рамки таблицы
        cellpadding - поля между границами ячейки и ее содержимым
        cellspacing - поля между ячейками
    
        Атрибуты таблиц, которые еще актуальные (для тегов th и td)

        colspan - указывает количество колонок, объединенных в одну ячейку
        rowspan - указывает количество строк, объединенных в одну ячейку
        headers - связывает данные ячейки с названиями соответствующих колонок по id
        scope (только для th) - указывает на ячейки колонок или строк таблицы
