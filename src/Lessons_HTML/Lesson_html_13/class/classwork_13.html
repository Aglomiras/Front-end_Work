Атрибуты select and option - для раскрывающихся списков

        select
        - name - определяет имя выпадающего списка
        - id - определяет идентификатор списка
        - multiple - позволяет выбрать несколько опций списка

        option
        - value - определяет значение опции, отправляемое на сервер
        - selected - определяет опцию, выбранную по умолчанию
        - disabled - определяет отключенную опцию

        button
        - type - опредяет тип кнопки (button(функции), reset(очистка), submit(отправка))
        - value - определяет значение кнопки
        - name - определяет название кнопки
        - disabled - указывает, что кнопка отключена
        - autofocus - устанавливает фокус на кнопке при загрузке страницы

        textarea
        - cols - определяет ширину текстового поля (кол-во символов)
        - rows - определяет высоту текстового поля (кол-во строк)
        - name - определяет название текстового поля
        - id - определяет идентификатор текстового поля
        - maxlength - определяет максимально разрешенное количество вводимых символов
        - autofocus - устанавливает фокус на текстовом поле при загрузке страницы
