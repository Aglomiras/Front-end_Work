video - тег добавления видео-файла

iframe - вставка другой веб-страницы

object - тег встраивания объекта на страницу

Любой элемент формата object необходимо заключить в обертку (div), так как они сами по себе являются строчными элементами, и чтобы они начинались с новой строки нужен div

embed - тег встраивания связанного ресурса
